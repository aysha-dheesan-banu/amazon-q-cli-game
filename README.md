## 🎮 What is Snake Evolution?

Snake Evolution is not just a single game - it's a complete **10-in-1 Educational Gaming Platform** that combines classic Snake gameplay with 10 different mini-games! Perfect for students of all ages to learn while having fun.

### 🎯 Main Features

- 🐍 **Classic Snake Gameplay** - Move, eat, grow!

- 🧮 **Educational Content** - Math, Science, Geography, History, Language
- 📈 **Progressive Difficulty** - Gets harder as you improve
- 🏆 **Score System** - Compete with friends
- 🌍 **Multilingual Support** - English and Tamil
- 📱 **Easy to Play** - Simple controls, engaging gameplay

## 🚀 Quick Start

### Requirements
- Python 3.8 or higher
- Pygame 2.5+

### Installation & Play

```bash
# Clone the repository
git clone https://github.com/aysha-dheesan-banu/amazon-q-cli-game.git
cd amazon-q-cli-game

# Install pygame
pip install -r requirements_snake.txt

# Run the game
python snake_evolution.py
```

### Alternative: Use the provided script
```bash
# Make script executable and run
chmod +x play_snake.sh
./play_snake.sh
```

## 🎮 How to Play

### 🐍 Main Snake Game
1. **Move the Snake**: Use arrow keys to control the snake
2. **Eat Food**: Guide the snake to eat the food items
3. **Answer Questions**: Solve educational problems for bonus points
4. **Grow & Score**: Snake grows longer, your score increases
5. **Avoid Walls**: Don't hit the walls or the snake's own body

### 🎯 Game Selection Menu
1. **Choose Category**: Press 1 for Educational Games or 2 for Python Games
2. **Select Game**: Pick from 5 educational or 5 arcade games
3. **Play & Learn**: Each game has unique gameplay and learning objectives
4. **Switch Games**: Return to menu anytime to try different games

### 🎮 Game Controls
- **Arrow Keys**: Navigate menus and control games
- **Number Keys (1-5)**: Select games from menu
- **Enter**: Confirm selections and submit answers
- **B**: Go back to previous menu
- **ESC**: Quit game
